Detailed Description

Freezing point tracker inheriting from the Chebyshev expansion root finder.

This class inherits the machinery needed to do root finding on the Chebysev expansion representing the temperature profile with depth, then implements the algorithm for tracking a freezing point as the temperature profile evolves
